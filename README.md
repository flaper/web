#Flaper
Based on angular2

##Requirements
* node 5.1
* bower (npm install bower -g)
* typings (npm install bower -g)

##Install
install packages  
`./init.sh`

##Config
If needed adjust file `src/app/config/config.local.json` with some settings from here - 
[src/app/config/config.json](src/app/config/config.json)

##Dev server

start the server  
`npm start`

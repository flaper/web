#Flaper
Based on angular2

##Install
install packages  
```bash
npm install  
bower install
```

install typescript typings  
`typings install`

##Config
Create file `src/app/config/config.local.json` with `{}` or override some settings from here - 
[src/app/config/config.json](src/app/config/config.json)
##Dev server

start the server  
`npm start`
